🌟 Why This Repository Stands Out

This Habit Tracker is designed to be beautiful, practical, and motivating.
It helps users:
	•	✅ Create and manage daily habits
	•	📅 Track task completion status
	•	⏰ Set duration and reminders
	•	🔥 Maintain streaks and best records
	•	📊 View weekly & monthly progress graphs
	•	🌙 Enjoy a premium dark-themed interactive UI
	•	📱 Use across desktop and mobile devices

✨ Key Features

📝 Habit Management
	•	Add habit name
	•	Add duration
	•	Add reminder time
	•	Mark task as completed / pending
	•	Delete or edit tasks

📊 Smart Analytics
	•	Daily progress tracking
	•	Weekly performance chart
	•	Monthly completion rate
	•	Streak counter
	•	Best streak record

🔐 Authentication
	•	Secure login/signup UI
	•	User-based habit storage
	•	Personalized dashboard

⸻

🛠️ Tech Stack

Frontend  : HTML, CSS, JavaScript
Backend   : Node.js / Express
Database  : MongoDB / Firebase
Charts    : Chart.js / Recharts
Auth      : JWT / Firebase Auth
